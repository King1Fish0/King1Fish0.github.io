The No Jump King

A game I created

Super Hidden Text


This project was started mostly to familiarize myself with 2D game engines, specifically godot. The gimmick of the game, being that you need to get around via teleportation, was inspired by a prompt my friend gave me. That being to create a 2D platformer without the ability to jump. The game spans 3 levels, consisting of ~20 minutes of gameplay, and is avaliable on itch.io.
